## 🔬 Models and Research

### Lotka-Volterra Model with Competition
- Classic predator-prey model
- Modifications with intra- and inter-specific competition
- Stability analysis of equilibrium points
- Bifurcation analysis

### Coupled Logistic Maps
- Systems of coupled maps
- Synchronization and desynchronization
- Spatiotemporal chaos
- Patterns in population dynamics

### Discrete Dynamical Systems
- Delay differential equations (with delay)
- Systems without delay
- Stability analysis of discrete systems
- Transitions to chaos via period-doubling bifurcation cascade

## 📊 Visualization

The repository includes tools for generating:

- **Phase portraits** — for continuous and discrete systems
- **Lameray diagrams** (cobweb plots) — iteration analysis of maps
- **Bifurcation diagrams** — studying system behavior dependence on parameters
- **Time series** — population dynamics over time
- **2D bifurcation diagrams** — dynamic regime maps
